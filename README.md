# text-figure宏包

## 编写背景

LaTeX的试卷排版中，题干和图片排版一直以来是一个比较麻烦的事情，于是想编写一个方便统一的命令用来排版这种题干和图片一起的情况。

## 宏包使用说明

提供`\textfigure[<key>=<val>]{<content>}{<figure or other content>}`

- 提供键值来控制
  - 图片相对于文本的位置
  - 图片的额外偏移
  - 整体的额外偏移
  - 文本的宽度
  - 文本占行距的比例

具体细节与示例欢迎用户查看用户手册。
